About

This application will track non-work/school related casual screen time.

How To Use

Simply log into the app any time you stop using a screen and log how long you've spent in front of the screen for that device.

Screens include items such as:

  • Desktop Computer
  • Laptop Computer
  • Game Console
  • Tablet
  • Phone

Feedback Provided

The application will provide access to records entered as well as graphs and advice based on the records entered.
